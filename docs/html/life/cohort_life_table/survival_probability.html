Survival probability

cohort_survival_probability(x, t, n=1, table='')

Compute the survival probability \(_np_x(t)\).

Note

The survival probability \(_np_x(t)\) is the probability that an individual aged \(x\) in calendar year \(t\) survives in the next \(n\) years. It is computed as

\[_np_x(t) = e^{-\int_0^n \mu_{x+\tau}(t+\tau) d\tau}\]

where \(\mu_x(t)\) is the force of mortality of an individual aged \(x\) in calendar year \(t\).

From a probabilistic point of view, \(_np_x(t)\) is the survival function of the random variable \(T_x(t)\) that is

\[_np_x(t) = Pr \left( T_x(t) > n \right) = Pr \left( T(t-x) > x+n \mid T(t-x) > x \right)\]

where \(T_x(t)\) is the random variable describing the remaining lifetime of an individual aged \(x\) in calendar year \(t\).

Finally, note that we have the following decomposition formula

\[_np_x(t) = {}_{\tau}p_x(t) \cdot {}_{n-\tau}p_{x+\tau}(t+\tau)\]

for all \(\tau \in (0,n)\).

Parameters

  • x (numeric) – Age.

  • t (numeric) – Calendar year.

  • n (numeric) – Horizon time.

  • table (string) – Life table to use.

Returns

Probability that an individual aged \(x\) in calendar year \(t\) survives in the next \(n\) years.

Return type

numeric (between 0 and 1)

Examples

Compute \(p_{50}(2030)\) according to the IABE table for males (probability that a Belgian man aged 50 in the calendar year 2030 will survive the following year):

>>> cohort_survival_probability(50, 2030, 1, "IABE male")
99.77%

Compare with a male aged 50 in the calendar year 2060:

>>> cohort_survival_probability(50, 2060, 1, "IABE male")
99.86%

Compute \(_{10.5}p_{54.5}(2030.75)\) according to the IABE table for females:

>>> cohort_survival_probability(54.5, 2030.75, 10.5, "IABE female")
96.01%
