Pure endowment

cohort_pure_endowment(x, t, n=1, i=0, table='')

Compute the present value of a pure endowment \(_nE_x(t)\).

Note

A pure endowment consists of paying an amount of 1 unit in \(n\) years from now to an individual currently aged \(x\) in calendar year \(t\) if the individual survives.

The present value is computed as

\[_nE_x(t) = \frac{_np_x(t)}{(1+i)^n}\]

where \(_np_x(t)\) is the probability that an individual aged \(x\) in calendar year \(t\) survives in the next \(n\) years and \(i\) is the guaranteed interest rate.

Parameters

  • x (numeric) – Age.

  • t (numeric) – Calendar year.

  • n (numeric) – Horizon time.

  • i (numeric) – Guaranteed interest rate.

  • table (string) – Life table to use.

Returns

Present value of the pure endowment.

Return type

numeric (non-negative)

Examples

Compute \(E_{50}(2030)\) according to the IABE table for males with \(i=2\%\):

>>> cohort_pure_endowment(50, 2030, 1, 0.02, "IABE male")
0.9782

Compare with a male aged 50 in the calendar year 2060:

>>> cohort_pure_endowment(50, 2060, 1, 0.02, "IABE male")
0.9790

Compute \(_{10.5}E_{54.5}(2030.75)\) according to the IABE table for females with \(i=2\%\):

>>> cohort_pure_endowment(54.5, 2030.75, 10.5, 0.02, "IABE female")
0.7799

Finally, note that by definition the calculation of a pure endowment amounts exactly to discounting a survival probability. For example, the result of the example above can be obtained as follows:

>>> cohort_survival_probability(54.5, 2030.75, 10.5, "IABE female") / (1 + 0.02)^10.5
0.7799
